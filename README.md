# IELTS Master

A 120-day IELTS preparation app built with React + DeepSeek AI.

## Features

- **Speaking** — AI examiner dialog with real-time scoring (Fluency / Lexical / Grammar / Pronunciation)
- **Reading** — 3-phase timed reading with vocabulary lookup
- **Writing** — Task 1 & Task 2 practice with AI feedback
- **Vocabulary** — Spaced repetition (SRS) with morpheme analysis and AI sentence generation
- **Progress** — 4-phase study plan tracking (Day 1–120)

## Getting Started

1. Get a DeepSeek API key at [platform.deepseek.com](https://platform.deepseek.com)
2. Clone this repo and run:

\```bash
npm install
npm run dev
\```

3. Enter your API key in the setup screen

## Study Plan

| Phase | Days | Target Score | Focus |
|-------|------|-------------|-------|
| Foundation | 1–30 | 5.0 | Vocabulary, Speaking Part 1, Free reading |
| Consolidation | 31–60 | 6.0 | Speaking Part 2, Timed reading |
| Breakthrough | 61–90 | 7.5 | Speaking Part 3, Writing, Advanced vocab |
| Final Sprint | 91–120 | 8.0 | Full mock tests, Weak point reinforcement |
